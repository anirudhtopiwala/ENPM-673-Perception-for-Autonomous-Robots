# Countingpins

This is program to count the number of pins in an image. 3 Different methods have been implemented to calculate the different colored pins in the image.

The function countcolors3 can also find out the white and the transparent pins.

Countcolor3 is different from the other methods, as here edge detection is used whereas in other methods boundaries are made and counted from binary images. 

TO run the code, run the main.m file.
